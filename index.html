Hexo搭建个人博客部署于Github

发表于 2017-08-18 | 分类于 Hexo
Hexo是一个轻量级的博客使用hexo生成静态博客并架设在免费的github page平台 系统: Window 使用软件: Git for windows Node.js for windows32位或64位按自己电脑的系统来选择，如果不清楚，可以直接在官网下载32位的，可以兼容64位系统 第一步：安装git软件。具体如何使用，请查找相关教程。第二步：安装Node.js软件。Hexo是靠node驱动，所以以上除了github以外都是是必须的。 第三步：安装hexo,在Git Bash端执行下面的命令就可以安装Hexo。 $ npm install hexo-cli -g 第四步：然后初始化hexo，进入文件之后就可以执行后面的命令了 $ hexo init blog $ cd blog 介绍一下hexo下用到的命令： $ hexo g/generate #生成静态文件 $ hexo s/server #启动服务器，主要用来本地预览 $ hexo d/deploy #将本地文件发布到github或Coding上 $ hexo n/new "postName" ...
