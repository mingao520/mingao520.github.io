Hexo搭建个人博客部署于Github

发表于 2017-08-18

Hexo是一个轻量级的博客
使用hexo生成静态博客并架设在免费的github page平台

系统:

  • Window

使用软件:

  • Git for windows
  • Node.js for windows
    32位或64位按自己电脑的系统来选择，如果不清楚，可以直接在官网下载32位的，可以兼容64位系统

第一步：安装git软件。具体如何使用，请查找相关教程。

第二步：安装Node.js软件。
Hexo是靠node驱动，所以以上除了github以外都是是必须的。

第三步：安装hexo,在Git Bash端执行下面的命令就可以安装Hexo。

 $ npm install hexo-cli -g

第四步：然后初始化hexo，进入文件之后就可以执行后面的命令了

 $ hexo init blog
  $ cd blog

介绍一下hexo下用到的命令：

 $ hexo g/generate #生成静态文件
$ hexo s/server #启动服务器，主要用来本地预览
$ hexo d/deploy #将本地文件发布到github或Coding上
$ hexo n/new "postName"#新建一篇文章
$ hexo n/new page "pageName" #新建页面
$ hexo h/help # 查看帮助
$ hexo v/version #查看Hexo的版本

第五步：依次执行命令行：

 $ hexo g
  $ hexo s

这时，正常情况下，用浏览器打开http://localhost:4000/，不出意外的话，你就可以看到你的博客了

第六步：你需要一个github的帐号，如何注册请查找相关教程。
1、点击”New repository”，新建一个版本库。
2、输入Repository name:yourname.github.io(yourname与你的注册用户名一致,这个就是你博客的域名了)

第七步：首先打开hexo文件的_condig.yml配置文件，拉到最下面，修改配置

 deploy:  
type: git
repository: https://github.com/yourname/yourname.github.io.git 
branch: master

第八步：安装插件

$ npm install hexo-deployer-git --save

第九步：上传文件至github.

$ hexo deploy

这时，如果正常https://yourname.github.io/ 就可看到你的博客了。

其他说明;

1、hexo 的博客文件都在\blog\source_posts下面，后缀为.md,需要markdown文件编辑器打开，可以新增、删除、修改。
2、本地查看方法：
先输入

$ hexo s

然后在浏览器输入：http://localhost:4000/

3、更新网站博客的方法：

$ hexo d -g
