折腾的HexoEditor安装过程

发表于 2018-04-15

由于之前利用hexo框架在github上搭建了一个静态网站，但是每次更新hexo网站，都比较麻烦，要用命令行进行更新、上传。最近在网上偶然遇到HexoEditor这个框架，可以很方便进行网站的更新、上传。
框架地址在：https://github.com/zhuzhuyule/HexoEditor
于是按照给出的流程进行安装：

//if use Windows:
npm config set prefix “C:/Program Files/nodejs/npm_global”
npm config set cache “C:/Program Files/nodejs/npm_cache”

//if use Linux\Mac:
npm config set prefix “~/nodejs/npm_global”
npm config set cache “~/nodejs/npm_cache”

//If In China, China, China, you can set mirror to speed up !
npm config set registry “https://registry.npm.taobao.org/“
npm config set electron_mirror “https://npm.taobao.org/mirrors/electron/“

git clone https://github.com/zhuzhuyule/HexoEditor.git
npm install -g electron@1.8.1
npm install
npm start

但是在最后一步，运行 npm start 时，出现错误：
npm start报missing script:start错误

在网上找了很久，终于发现，这个给出流程出现了一个比较重大的遗漏，就是要安装HexoEditor！

解决方法是：
1、找到HexoEditor所在的目录，一般在电脑个人帐号下的目录下
2、在该文件夹下运行power shell命令行：npm install
3、然后运行命令：npm start

在HexoEditor文件夹下安装后，运行正常。

# hexo
